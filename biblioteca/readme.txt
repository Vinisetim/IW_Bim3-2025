Criar um sistema para acessar os livrros cadastrados em uma biblioteca virtual. Para isso, precisamos cadastrar os objetos da nossa biblioteca virtual:

- Nome para a biblioteca
- Endereço para a biblioteca

um array chamado livros e vamos cadastrar três objetos para cada livro:
-titulo
-autor 
-ano

Vamos criar uma função que irá percorrer o array dos livros cadastrados. Estas função será chamada por meio de um botão que teremos na nossa tela frontend - HTML.

Vamos precisar :

Um arquivo HTML
Um arquivo cadastrados
Um arquivo Javascript
